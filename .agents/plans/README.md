# 🧠 Brain Sync Project Map (MOC)

이 프로젝트는 브라우저 대화를 아카이빙하고, RAG를 통해 나만의 지식 베이스를 구축하는 것을 목표로 함.

## 🗺️ Implementation Map
- [[architecture|🏗️ 시스템 아키텍처]]: 전체 서비스 구조 및 기술 스택
- [[data-flow|🔄 데이터 흐름]]: 수집부터 검색까지의 여정
- [[roadmap|🚀 개발 로드맵]]: 현재 단계와 다음 스텝

## 🛠️ Components
- **Extension**: 웹 페이지 데이터 추출 및 백엔드 전송
- **Backend (FastAPI)**: 데이터 처리, 요약, 벡터화 및 DB 관리
- **Vector DB (Qdrant)**: 시맨틱 검색을 위한 벡터 저장소
- **RDB (PostgreSQL)**: 원본 데이터 및 메타데이터 보관
- **Web (Next.js)**: 지식 탐색 및 관리를 위한 대시보드
